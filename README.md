# coherence-complexity
Reference simulations and visualizations for the Coherence Complexity (Cₖ) framework – a structural measure of integration effort in adaptive state spaces.

## Core formulation

Ck(x) = α D(x, IRC) + β B(x)

dx/dt ≈ −∇Ck(x)

where

D(x,IRC) = distance to the reference integration core  
B(x) = barrier field representing structural constraints
